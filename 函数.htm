函数必须含有返回值

  • 定义函数的命名格式如下：
    create or replace function [函数名称]
    return [数据类型]
    [as|is]
    --变量声明
    begin
    --执行语句
    exception
    --异常处理
    end;
  • 执行函数可以使用SQL语句
    select [函数名] from dual;
  • 执行函数也可以使用匿名的PL/SQL块
    begin
     dbms_output.put_line([函数名]);
    end;
  • 在JDBC中调用函数，使用CallableStatemnent对象
